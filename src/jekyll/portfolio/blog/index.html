Node Blog CMS

Interactive CMS Built With JSNode And MongoDB

This website allows a user to fully manage it's content and appearance, create publications, change background color, text elements color, banner on the landing page, register and remove users/admins.
This web application is using MongoDB (non-sequel database) to store the data saved by a user, so basically this app is the "Content Management System" (CMS) that is provides a lot of UI flexibility for a user that has no coding skills.
Key Features:
  • User (with "Admin" privelege) can change page background color,
  • User (with "Admin" privelege) can manage most of the text elements color,
  • User (with "Admin" privelege) can change or completely remove a banner from the home page,
  • User (with "Admin" privelege) can change Title of the website and Slogan as well,
  • Any of the UI changes can be previewed first,
  • User can create/edit/remove his/her publications/posts,
  • If user is an "Admin", he/she can manage other users' publications as well as add or remove other users from the database
