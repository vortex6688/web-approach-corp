PUNKIN THE BAND!

This website represents "PUNKIN THE BAND!", a very unique music band.
The website contains the band's music tracks, videos, other related material. The website's costom design was provided by client and updated by Web Approach, all the code and mobile friendly interface also developed by the Web Approach Corp.
Link to the live website: http://www.punkintheband.com
This website allows a user to fully manage it's content and appearance, create/edit/remove publications by using user friendle integrated text/content editor, which is also provides a lot of flexebility to customize the publications elements in many different ways.
This web application is using MongoDB (non-sequel database) to store the data saved by a user, so basically this app is the "Content Management System" (CMS) that is provides a lot of UI flexibility for a user that has no coding skills whatsoever.
